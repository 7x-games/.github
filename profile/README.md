## 🎯 What Is 7x.games?

7x.games is a collection of **150 free browser games** built for speed, simplicity, and zero friction. Every game loads instantly, works on mobile and desktop, and requires no account or download.

Built solo by **[Sushil Namberdar](https://github.com/sushilnamberdar)** — one developer, one goal, 150 games.
